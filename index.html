Makes the Dark Bramble less scary. Options to remove scary elements individually.
TAICheat
Collection of cheats. Super jet pack, invincibility, and a lot more.

Become a modder

If you want to make your own mods, the OWML documentation has most of the info you need to get started. The easiest way to start is by cloning the mod template project and following the instructions there. The aforementioned Discord server also has a modding channel.
